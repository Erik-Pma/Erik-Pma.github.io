About Me

My name is Erik Praitis. I am a game developer that has been working on making game from 2021. I staretd by doing gamejams and even got to work on a cross platform MMO called Castaways. I specialise on making designer friendly, and modular game systems such as, a borderlands style weapon generator, an ai behaviour tree, an in depth movement system with climbing, slideing and wall running. On Castways I helped write calls to let players purachase skins and store them in a server side database so let players equip them as well as make it so the store page runs off a yaml file to update the store at anytime with out having to push a big update to remove items that may be broken, I also helped write networked projectiles into the game so players can interact with eachother, and helped get the game into a stable state for users to play.
